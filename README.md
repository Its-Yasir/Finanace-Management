# Personal Expense Tracker (PET)

A modern, responsive web application for tracking personal expenses built with vanilla JavaScript and Firebase.

## About

Personal Expense Tracker (PET) is a user-friendly financial management tool that helps you monitor your income and expenses in real-time. With beautiful charts, category-based tracking, and a clean interface, managing your finances has never been easier.

## How It Works

### Getting Started

1. **Create an Account** - Register with your email and password
2. **Add Your Income** - Start by adding your income sources (salary, freelance, etc.)
3. **Track Expenses** - Add expenses as you spend with amount, category, and date
4. **View Analytics** - See your spending patterns through interactive charts and reports
5. **Manage Your Profile** - Update your settings and preferences anytime

### Main Features

**Dashboard**

- View your financial overview at a glance
- See total income, expenses, and current balance
- View recent transactions
- Quick access to add income or expenses

**Expense Management**

- Add new expenses with amount, category, date, and description
- Edit or delete existing expenses
- Filter expenses by category
- View all expenses in an organized table

**Reports & Analytics**

- Pie chart showing expense distribution by category
- Bar chart displaying monthly spending trends
- Category breakdown with percentages
- Summary statistics including average monthly spending

**Settings**

- Update your profile information
- Change your email or password
- Manage account preferences
- Toggle between light and dark themes
